## 🤖 AI分析引擎

本项目配套 [lottery-analyzer](https://github.com) skill：

- 每天自动抓取竞彩网实时赔率
- 搜索球队伤病/阵容/状态
- 生成每场全玩法（SPF+让球+总进球+半全场+比分）分析
- 输出500/1000元两档投注方案
