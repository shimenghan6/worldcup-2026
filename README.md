## 🚀 一键安装

双击 `setup_scheduler.bat`（Windows），自动完成三件事：

1. `pip install requests` — 装依赖
2. 创建 Windows 定时任务 — 每小时自动抓取
3. 试跑一次 — 验证能否正常抓数据

之后全自动，不需要再操作。

### 手动运行

```bash
python fetch_odds.py              # 抓取 + 更新 + 推送
python fetch_odds.py --dry-run    # 仅测试
```

### 依赖

```
pip install requests    # 唯一依赖
```

## 🛡️ 数据校验

每次更新后自动运行 `node validate-data.js`，逐场校验 5 条规则：

```
tip='胜' → 主队赢 → 比分主>客（如 2:0）
tip='负' → 客队赢 → 比分主<客（如 0:2）
tip='平' → 打平   → 比分相等（如 1:1）
tip='胜' → 半全场不能含 负-负
tip='负' → 半全场不能含 胜-胜
```

校验不通过 → 不推送。
